Hash Function

发表于 2021-07-17 更新于 2021-07-30 分类于 ACM

https://ac.nowcoder.com/acm/contest/11166/H

解题思路：

将数组先从小到大排序。然后枚举答案，若是x对于a[i]符合条件，那么之后只要统计a[i+1]~n的情况了，如果发现有遇到了重复的，则说明当前这个答案不可以，继续遍历。

还有一种解法，答案不能使任意两个数的差的倍数。那么可以使用FFT，多项式相乘，对于系数>0的情况，说明这个指数不可以，但因为是相减，所以需要加 一个偏移量。接着枚举，为某个已出现的指数的倍数便不可以。
